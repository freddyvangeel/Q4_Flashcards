# Q4 Flashcards

Streamlit app die willekeurig wetsartikelen als flitskaart toont.

## Gebruik

1. Plaats het bestand `Juridisch kader Q1 tm Q5.md` in deze repo (volledige inhoud).
2. Installeer dependencies:
   ```bash
   pip install streamlit requests beautifulsoup4
   ```
3. Start de app:
   ```bash
   streamlit run app.py
   ```

## Werking

- Gebruikt alleen regels met **Artikel** (geen complete wetten/regelingen)
- Voorkant = omschrijving
- Achterkant = exacte wettekst via wetten.overheid.nl
